# flutter_amazon_connect_webrtc — example

A minimal Flutter app that starts an audio or video call into Amazon Connect using the plugin.

## Run

The platform folders (`ios/`, `android/`) are generated by Flutter. From this directory:

```bash
flutter create .                    # generate ios/ + android/ runners (first time only)
flutter pub get

# Run against your backend (DEMO_JWT optional — only if you front the API with auth):
flutter run \
  --dart-define=BACKEND_BASE_URL=https://<apiId>.execute-api.eu-west-2.amazonaws.com/v1 \
  --dart-define=DEMO_JWT=<optional-bearer-token>
```

## Required platform configuration

After `flutter create .`, add the media permissions the Chime SDK needs.

### iOS — `ios/Runner/Info.plist`
```xml
<key>NSMicrophoneUsageDescription</key>
<string>Voice calls with support</string>
<key>NSCameraUsageDescription</key>
<string>Video calls with support</string>
<key>UIBackgroundModes</key>
<array>
  <string>audio</string>
  <string>voip</string>
</array>
```
Set the iOS deployment target to **15.0** in `ios/Podfile` (`platform :ios, '15.0'`).

**Required — enable the permission handlers.** `permission_handler` compiles every permission OUT
unless its macro is defined, returning *denied with no prompt*. Add to the Podfile `post_install`:
```ruby
installer.pods_project.targets.each do |target|
  target.build_configurations.each do |config|
    config.build_settings['GCC_PREPROCESSOR_DEFINITIONS'] ||= ['$(inherited)']
    config.build_settings['GCC_PREPROCESSOR_DEFINITIONS'] << 'PERMISSION_MICROPHONE=1'
    config.build_settings['GCC_PREPROCESSOR_DEFINITIONS'] << 'PERMISSION_CAMERA=1'
  end
end
```

### Android — `android/app/src/main/AndroidManifest.xml`
The plugin already declares `RECORD_AUDIO`, `CAMERA`, `FOREGROUND_SERVICE*`, etc. Ensure the app's
`minSdkVersion` is **24+** and `compileSdkVersion`/`targetSdkVersion` are **34** in
`android/app/build.gradle`, with Java 17.

## What it shows

- Start an **audio** (VoIP) or **video** (WebRTC) call.
- Live `CallState` via `ValueListenableBuilder`.
- Mute / unmute, enable video, switch camera, end call.
- Local + remote video tiles via `ConnectVideoView` (PlatformView).
- Context `{issueType: billing, tier: gold}` is sent so Connect routes to the right queue.
